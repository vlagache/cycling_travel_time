Cycling Travel Time

Cycling Travel Time est un projet étudiant, dont le but est de prédire une durée de déplacement à vélo par l'entrainement d'un modele de Machine Learning sur les données d'entrainements passés. L'utilisation de cette application necessite d'avoir un compte Strava sur lequel des activités ont été enregistrés.
